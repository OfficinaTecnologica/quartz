Per capire come usare un comando posso usare `man (nome del comando)`

al posto di NOME DEL COMANDO devi mettere il nome del comando. Sì, esatto, ho dovuto specificarlo

**Muoversi nel file system** (So che voi scrollatori seriali non avete mai visto una cartella (directory) o un file ma oggi diventerete degli uomini, o delle donne, a vostra preferenza)
- elencare i file in una directory: `ls`
- come capire in che cartella sei `pwd`
- cambiare directory: `cd (percorso della directory)`
- andare alla home: `cd`
- andare alla directory un livello sopra `cd ..`
- copiare un file `cp (nome file) (nome nuovo file)`
- cancellare un file `rm`
- spostare o rinominare un file `mv`
- creare una directory

**Installare programmi**
- per cercare un programma: `dnf search`
- per installare un programma: `sudo dnf install (nome del programma)`
	al posto di (nome del programma) devi mettere il nome del programma, senza parentesi
- per rimuovere un programma: `dnf remove (nome del programma)`
